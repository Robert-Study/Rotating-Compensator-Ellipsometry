# Ellipsometry Analysis

This repository contains the Python analysis code I developed for the **ellipsometry section** of my **third-year university group project**, which I also **led as project leader**.

**Report link:** <u>Awaiting permission to share — please check back on April 26.</u>

The workflow was built to take **raw rotating-compensator ellipsometry measurements** through to:

- fitted **harmonic coefficients**
- extracted **Psi** and **Delta**
- **instrument calibration**
- **thin-film simulation**
- final estimation of **film thickness** and **optical constants**

---

## What the code does

The repository is split into separate files covering the main stages of the analysis:

- **Data import and preprocessing**
- **Harmonic fitting of measured intensity sweeps**
- **PCSA instrument modelling**
- **Fresnel--Airy thin-film simulation**
- **Psi and Delta extraction**
- **Instrument calibration**
- **Film property fitting**
- **Plotting and output generation**
- **End-to-end workflow execution**
