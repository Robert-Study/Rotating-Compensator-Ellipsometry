This repository contains the Python analysis code I developed for the ellipsometry section of a my third year university group project led by myself.

The workflow was built to take raw rotating-compensator ellipsometry measurements through to fitted harmonic coefficients, extracted Psi and Delta, perform calibration, thin-film simulation, and final estimation of film thickness and optical constants.

## What the code does

The repository is split into separate files covering the main stages of the analysis:

- **data import and preprocessing**
- **harmonic fitting of measured intensity sweeps**
- **PCSA instrument modelling**
- **Fresnel--Airy thin-film simulation**
- **\(\Psi\) and \(\Delta\) extraction**
- **instrument calibration**
- **film property fitting**
- **plotting and output generation**
- **end-to-end workflow execution**

This project demonstrates:
